## ­Ъћї API Fallback рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў

`api.ts` рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА backend-рЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљрЃА:

1. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА `/health` endpoint-рЃА
2. рЃЌрЃБ backend рЃЏрЃўрЃБрЃгрЃЋрЃЊрЃЮрЃЏрЃћрЃџрЃўрЃљ, рЃњрЃљрЃЊрЃљрЃћрЃарЃЌрЃЋрЃћрЃЉрЃљ mock data-рЃќрЃћ
3. рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ API call рЃЕрЃљрЃЋрЃљрЃарЃЊрЃћрЃЉрЃљ, рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃА mock-рЃА

```typescript
// API рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў
apiService.isUsingMockData()  // рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА mock mode-рЃА
apiService.enableMockData()   // рЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃА mock mode-рЃА
apiService.disableMockData()  // рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА backend-рЃќрЃћ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљрЃА
```

## ­ЪДф рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў

### рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў

- рЃДрЃЋрЃћрЃџрЃљ рЃарЃћрЃАрЃбрЃЮрЃарЃљрЃюрЃА рЃљрЃЦрЃЋрЃА `isActive: true`
- рЃЏрЃћрЃюрЃўрЃБ рЃЊрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ

## РџА Backend Developer-рЃўрЃАрЃЌрЃЋрЃўрЃА

### API Endpoints рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃБрЃюрЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ:

#### Restaurants
- `GET /restaurants` - рЃърЃљрЃњрЃўрЃюрЃљрЃфрЃўрЃљ, рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ
- `GET /restaurants/:id`

#### Menu Items
- `GET /menu-items` - restaurantId рЃцрЃўрЃџрЃбрЃарЃў
- `GET /menu-items/:id`

#### Orders
- `GET /orders` - userId рЃцрЃўрЃџрЃбрЃарЃў
- `GET /orders/:id`
- `POST /orders`
- `PATCH /orders/:id/status`
- `GET /orders/:id/tracking`
- `PATCH /orders/:id/assign-courier`

#### Auth
- `POST /auth/send-verification-code`
- `POST /auth/verify-code`
- `GET /auth/profile`
- `POST /auth/verify-token`
- `POST /auth/complete-registration`

### Response Format
```typescript
{
  data: T[] | T,
  total?: number,
  page?: number,
  limit?: number
}
```

### рЃњрЃљрЃЏрЃЮрЃарЃЌрЃЋрЃљ
рЃарЃЮрЃфрЃљ backend рЃЏрЃќрЃљрЃЊ рЃўрЃЦрЃюрЃћрЃЉрЃљ:
